# addClassMethod

Adds a new class method model to an existing class.

## Parameters

| Parameter Name | Type | Description |
| :--- | :--- | :--- |
| `classItemId` | number | The ID of the class item. |
| `methodName` | string | The name of the method. |
| `isAsync` | boolean | Is the method asynchronous? |
| `returnTypeId` | object | The ID of the return type. |
| `returnClassId` | object | The ID of the return class. |
