# addOrgDesk

Adds a new Org desk to the specified Org chart.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `orgChartId` | number | The ID of the Org chart to add the desk to. |
| `deskName` | string | The name of the new Org desk. |

## Example

```json
{
  "orgChartId": 1,
  "deskName": "Engineering"
}
```
