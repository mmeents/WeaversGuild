# listProjects Command Documentation

## Overview
The `listProjects` command retrieves a list of all root level projects in the system.

## Parameters
This command takes no parameters.

## Usage
Call the `listProjects` command to get a comprehensive list of all root-level projects currently defined in the graph.

## Example

```
listProjects
```

## Output Description
The command returns a list of project objects, each containing details such as project ID, name, and other relevant metadata.

## Notes
- This is a utility command useful for identifying project IDs needed for other operations.
- It provides a high-level view of the project structure without diving into nested folders or solutions.
