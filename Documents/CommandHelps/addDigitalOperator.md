# addDigitalOperator

Adds a digital operator to the specified DigitalOperatorPoolModel parentItem.

## Parameters

- **parentItemId**: (number) The ID of the parent DigitalOperatorPoolModel.
- **operatorName**: (string) The name of the new digital operator.

## Usage

This command is used to extend a digital operator pool by adding a new named operator. It requires the ID of the existing pool (parent item) and the desired name for the new operator.
