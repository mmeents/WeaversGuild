# addHtmlFile

Adds a new .html file item in the specified folder item, infra adds ext to name.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `folderItemId` | number | The ID of the folder item where the new HTML file will be created. |
| `fileName` | string | The name of the new HTML file (extension will be added automatically). |
| `fileContent` | string | The content of the HTML file. |
