## Transform to Outline 

A technique where a layout entire design is expected so need to keep inputs to a minimum.

## Default expectations

1. All tables have a default Id column.
2. Using Id in a name is reserved for FK to a column. example: x.itemId -> item.Id
3. References to an item may not have an Id yet. When so use omit Id: part.
example: Id:Name:Type or Name:Type
4. Leave Index details off.
